# Desert Thunder's Dotfiles

This book collects notes and guides for my personal NixOS and dotfiles setup.

The config reflects how I actually work. Some files stay long until they become
painful enough to split. The Neovim config comes from an older setup and still
carries that history.

If you spot something broken or confusing, open an issue.

For system administration and multi-machine setup details, see the
[NixOS](./docs/src/nixos.md) and [Guides](./docs/src/guides.md) docs.
