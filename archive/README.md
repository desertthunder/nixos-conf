# Archive

Contains old dotfiles & configs as references for wiring flakes.

## Purpose

This directory serves as a historical reference and migration guide for transitioning from traditional dotfile management to Nix-based configuration.

## Contents

- **Legacy configurations**: Pre-Nix dotfiles and setup scripts
- **Migration references**: Examples of how traditional configs map to Nix expressions
- **Historical context**: Evolution of the personal development environment

## Usage

Use these files as reference when:
- Setting up similar configurations in Nix
- Understanding the migration path from traditional setups
- Needing examples of non-Nix configuration patterns

## Note

Files in this directory are no longer actively maintained but preserved for their reference value in understanding the transition to declarative configuration management.
