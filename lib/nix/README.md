# Nix

NixOS configuration lives here.

- `hosts/` contains machine-specific NixOS entries.
- `modules/nixos/` contains shared system modules.
- `modules/home-manager/` contains shared Home Manager modules.
- `profiles/` is reserved for future role-based composition.

- `owais-nix-hp`
